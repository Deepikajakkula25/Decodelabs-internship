# Password Generator Application

## Project Description

The Password Generator Application is a Python-based command-line program that generates secure and random passwords based on user-defined requirements. The application helps users create strong passwords using a combination of uppercase letters, lowercase letters, numbers, and special characters to improve account security.

This project was developed as part of the Decode Labs Python Internship Program to demonstrate the use of Python's randomization techniques, string manipulation, and user interaction.

## Features

- Generate secure random passwords
- User-defined password length
- Includes uppercase and lowercase letters
- Includes numbers and special characters
- Fast and easy-to-use interface
- Enhances password security
