# Expense Tracker Application

##  Project Description

The Expense Tracker Application is a Python-based command-line program designed to help users record, manage, and analyze their daily expenses. The application allows users to add expense amounts, view expense history, and calculate the total amount spent, providing a simple solution for personal expense management.

This project was developed as part of the Decode Labs Python Internship Program to demonstrate fundamental Python programming concepts, including data handling, mathematical operations, loops, conditional statements, and accumulators.

## Features

- Add new expenses
- View complete expense history
- Calculate total spending automatically
- Menu-driven user interface
- Simple and efficient expense management
- Real-time data processing
