# General Knowledge Quiz Application

## Project Overview

The General Knowledge Quiz Application is a Python-based console program designed to test a user's general knowledge through a series of questions. The application evaluates user responses, calculates a score, and provides feedback based on performance.

This project was developed as part of the Python Programming Industrial Training Program to demonstrate fundamental programming concepts and problem-solving techniques.

## Features

- Interactive command-line interface
- Multiple general knowledge questions
- User input handling
- Answer validation using conditional statements
- Real-time score calculation
- Final performance summary
- Beginner-friendly implementation

## Technologies Used

- Python 3
- Visual Studio Code (VS Code)
- Git
- GitHub

## Python Concepts used

- Variables and Data Types
- User Input 
- Conditional Statements 
- String Manipulation 
- Boolean Logic
- Score Tracking
- Program Flow Control

## How It Works

1. The program displays a question.
2. The user enters an answer.
3. The answer is compared with the correct answer.
4. The score is updated if the answer is correct.
5. After all questions are completed, the final score is displayed.
