# To-Do List Application

## Project Overview
This is a simple command-line To-Do List Application developed using Python. The application helps users manage their daily tasks efficiently by allowing them to add, view, and delete tasks.

## Features
- Add new tasks
- View all tasks
- Delete completed or unwanted tasks
- Save tasks permanently using file handling
- User-friendly menu-driven interface

## Technologies Used
- Python
- VS Code
- Git
- GitHub

## Concepts Used
- Lists
- Functions
- Loops
- Conditional Statements
- File Handling
- Exception Handling
